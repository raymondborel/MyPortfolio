Restaurant Recs

Restaurant Recs is a MERN stack web application that shows users a list of recommended restaurants created by other users. We implemented Yelp's restaurant API for this project for users to search reviews.

Pair Matching Game

For my first project I created this simple card matching game. Users start by clicking new game where each card would get initialized a pair value. Then users would click a card and try to find its match until all matches are found.
